Adems, el resto de datos de la columna se han interpretado como texto. Usa la siguiente funcin para crear una columna de nombre precipitationN que contendr los valores numricos.

Lo que vamos a hacer es transformar en un nmero si encontramos un valor numrico, y en el caso de que encontremos T, el valor correspondiente ser 0.00. El valor T significa que en ese da hubo trazas de lluvia. Es decir, llovi, pero muy poco, y no ha quedado registrado el volumen de precipitacin.

In [24]:
def precToN(p):
    if p == 'T':
        return 0.0
    else:
        return double(p)
In [25]:
d['precipitationN'] = d.precipitation.apply(lambda x: precToN(x))
